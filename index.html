The grantcam uses a Connectix Color QuickCam to broadcast the view of my desk to the masses. It is connected to a vintage Macintosh Quadra 950 with a minimal 28 mb of RAM, Mac OS 8.1 and Open Transport 1.3 - if you can believe it. Up until July 1999, I was using an original greyscale QuickCam. Rather than retire this pixel-pushing workhorse, I am going to use it as an alternate viewpoint camera for this site.

Images are captured and streamed as JPEG formatted data using Rearden SiteCam and served up with a StarNine WebStar server. Not bad for a eight year old Macintosh! Just try doing this with an Intel box this old. Ha!

If you are interested, find out just what the grantcam is good for. An exceptional friend of mine proves that the grantcam isn't just a blatantly gratuitous pile of technology. It's practical too.

On a good day, you may be able to get up to a one frame every couple of seconds refresh rate on the streaming image. The faster your connection, the faster the updating. An average connection speed will yield only about one frame every four or five seconds. The current limitation on the refresh speed is not my internet connection, but the speed at which the serial port that the QuickCam is plugged into can pump data into the Mac. Someday, when I get ahold of a dedicated Nubus video capture card, then we'll start to see some speed.

Oh yeah, I also run a copy of the Rumpus ftp server from Maxum Development Corporation on this poor boy. This allows remote editing of html files via ftp using my editor of choice, BBEdit from Bare Bones Software. All this and I still have over 6 mb of RAM left over. Almost enough to run a trimmed down web browser.

Previously, the grantcam ran for nearly two years on a Macintosh IIcx with only 8 mb of RAM. Yes, you read that right, a ten year old 68020 processor! This particular box used System 7.1 and MacTCP version 2.0.4, while images were captured and served using a combination of WebCamToo and MacHTTP, the precursor to WebStar originally developed by Biap Systems.

Besides being the developer of Rumpus and other very cool web tools, Maxum Development also sponsors the Serve Different campaign. You too should use a Mac to dish out your web content responsibly.

If you're even a fraction of the typography nut that I am, you'll appreciate knowing a little bit about the fonts used on this site. The headlines are set in 18 point eWorld Tight, a bitmap font designed for Apple Computer's ill fated online service. If you're running a Macintosh, and I hope you are, the body copy should appear as 9 point Geneva, also an optimized bitmap face.

Since Geneva is a font that has been burned into the ROMs of every Macintosh since the beginning of time, this seemed like a good choice when using the <font face> tag. Besides its ubiquity, Geneva is one of the cleanest looking onscreen faces I have every seen, especially at smaller sizes.

If you're running Windows or some convoluted Unix machine, the body copy will appear as Arial or some other reasonably ugly default font that your browser has specified.
